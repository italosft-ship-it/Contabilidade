SITE NASSER LIMA CONTABILIDADE - VERSÃO PREMIUM COM IMAGENS

Arquivos principais:
- index.html = página principal do site
- style.css = visual do site
- assets/ = imagens e ícones locais em SVG
- posts/ = artigos separados em HTML

Como usar no GitHub Pages:
1. Crie um repositório no GitHub.
2. Faça upload de todos os arquivos e pastas mantendo a estrutura.
3. Em Settings > Pages, publique a partir da branch main e pasta root.
4. Troque o número fictício do WhatsApp no index.html e nos posts.

Para adicionar uma nova postagem:
1. Copie um arquivo da pasta posts.
2. Renomeie, por exemplo: obrigacoes-mei.html
3. Troque título e texto.
4. Adicione um novo bloco de link na seção Artigos do index.html.

Observação:
As imagens foram feitas em SVG para o site ficar leve, rápido e com visual profissional.
